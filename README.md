# cmd-ai

Крошечная «нейросеть в терминале» — отправь запрос через curl и получи ответ прямо в CMD/Terminal. Идеально для чего? просто так.


## Что это

Простой API-обёртка, которая принимает вопрос в параметре `q` и возвращает текстовый ответ от модели.

## Пример использования

1. В терминале:
```bash
curl -k "https://cmd-ai.vercel.app/api/ask?q=как+дела+друг"
