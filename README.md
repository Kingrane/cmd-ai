# cmd-ai

Нейросеть в терминале с сложным запуском :) — отправь запрос через curl и получи ответ прямо в CMD/Terminal. Идеально для чего? просто так.


## Что это

Простая API-обёртка, которая принимает вопрос в параметре `q` и возвращает текстовый ответ от модели.

## Пример использования

В терминале:
```bash
curl -L cmd-ai.vercel.app/how+many+days+in+one+year
```
если хочется пробелы вместо плюсиков :D
```
curl -L -G "cmd-ai.vercel.app" --data-urlencode "q=how many days in one year"
```
