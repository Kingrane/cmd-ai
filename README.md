# cmd-ai

Нейросеть в терминале с сложным запуском :) — отправь запрос через curl и получи ответ прямо в CMD/Terminal. Идеально для чего? просто так.


## Что это

Простой API-обёртка, которая принимает вопрос в параметре `q` и возвращает текстовый ответ от модели.

## Пример использования

1. В терминале:
```bash
curl -L cmd-ai.vercel.app/api/ask?q="how+are+you"
```
если хочется пробелы вместо плюсиков :D
```
curl -L -G "cmd-ai.vercel.app/api/ask" --data-urlencode "q=how many days in one year"
```
